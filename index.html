📁 Make macros global (PERSONAL.XLSB)

      
Want these scripts available in any Excel file without copying every time? Add them to your Personal Macro Workbook.

      
    
        
  1. Open any Excel file. Press Alt + F11 to open VBA editor.
    2. 
        
  2. If PERSONAL.XLSB does not exist: close VBA editor, go to View → Macros, type any name (e.g. "CreatePersonal"), click Create. Then close Excel and save when prompted. Reopen Excel → PERSONAL.XLSB is created.
    3. 
        
  3. In VBA editor (Alt+F11), locate VBAProject (PERSONAL.XLSB) in the Project Explorer.
    4. 
        
  4. Right-click → Insert → Module. Paste the full script (e.g., CalculateReceivings).
    5. 
        
  5. Repeat for each macro: GenerateDailyReport, GenerateWeeklyReport (each in separate module or same).
    6. 
        
  6. Press Ctrl+S to save PERSONAL.XLSB. (If protected: enable trust access to VBA project under Macro Settings).
    7. 
        
  7. Now you can run CalculateReceivings, GenerateDailyReport, GenerateWeeklyReport from Alt+F8 in any Excel workbook!
    8. 
      

      

        💡 Pro tip: Assign shortcut keys by opening Macro dialog → Options → set Ctrl+Shift+letter. Your global macros become instant productivity boosters.
      

    


    
⚠️ Enable macros — only for trusted files.
